# 数字化服务全成本测算系统

本项目用于建设一个轻量化的数字化服务全成本测算工具，核心目标是将 Excel 表格中的收入、成本、税金、利润率等计算逻辑沉淀为可配置、可复用、可留痕的测算系统。

## 第一阶段目标

第一阶段先实现本地单页面 Web 版 MVP：

- 新建、编辑、复制测算方案
- 配置收入、成本、税率、期间费用等参数
- 自动计算含税金额、不含税金额、全成本、利润和利润率指标
- 支持成本明细增删改
- 支持测算结果本地保存
- 支持导出 Excel 测算表
