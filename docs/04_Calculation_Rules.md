# 计算规则文档

## 1. 计算口径说明

本文档用于定义数字化服务全成本测算系统的核心公式。后续代码实现应以本文档为准。

第一阶段计算目标是复刻现有 Excel 表格的主要逻辑，并将公式参数化。

## 2. 金额字段定义

### 2.1 含税金额

含税金额是业务录入或合同口径金额。

### 2.2 不含税金额

不含税金额用于利润测算。

公式：

```text
不含税金额 = 含税金额 / (1 + 税率)
```

当税率为 0 时：

```text
不含税金额 = 含税金额
```

## 3. 收入计算

### 3.1 单条收入不含税金额

```text
收入不含税金额 = 收入含税金额 / (1 + 增值税率1)
```

增值税率1统一取参数配置，不在收入明细行单独维护。

### 3.2 收入合计

```text
含税销售收入 = 所有收入项含税金额之和
不含税销售收入 = 所有收入项不含税金额之和
销项税额 = 含税销售收入 - 不含税销售收入
```

## 4. 成本计算

### 4.1 手动录入金额

适用于直接录入含税金额的成本。

```text
含税成本 = 手动录入金额
不含税成本 = 含税成本 / (1 + 进项税率)
```

进项税率统一由参数配置和成本项中的“税率选择”带出，不在成本明细行直接输入。第一阶段规则为：产品分摊成本、基础设施成本、交付实施成本、定制开发成本、运维成本可选增值税率1、增值税率2、自有人工税率、外包税率；期间费用可选管理费用比例、销售费用比例。

### 4.2 单价乘项目数量

适用于外包、平台服务等按项目数量计算的成本。

```text
含税成本 = 单价 * 项目数量
不含税成本 = 含税成本 / (1 + 进项税率)
```

说明：

- 人天、合同数量、项目数量等统一使用“数量”字段表达。
- 自有人工可用“单价 x 数量”计算，其中数量可填写人天数。
- 自有人工默认选择自有人工税率，默认值为 0。

### 4.3 按收入比例计算

适用于期间费用中的管理费用、销售费用。

```text
费用金额 = 合同额 * 费用比例
```

费用比例不在明细行中手动填写，而是从参数配置中的管理费用比例、销售费用比例中选择。

合同额按含税销售收入合计取值。第一阶段期间费用的方式列只允许选择管理费用比例或销售费用比例。

第一阶段默认期间费用按合同额计提。

## 5. 成本分类汇总

### 5.1 分类成本小计

```text
分类含税成本 = 分类下所有成本项含税金额之和
分类不含税成本 = 分类下所有成本项不含税金额之和
```

### 5.2 项目全成本

```text
项目全成本 = 产品分摊成本 + 基础设施成本 + 交付实施成本 + 定制开发成本 + 运维成本 + 期间费用 + 模拟计息财务费用 + 附加税金
```

系统同时保留含税金额口径和不含税金额口径。

## 6. 附加税金

附加税金可按销项税额或不含税销售收入计提。结合截图表格中附加税金数值，第一阶段建议做成可配置公式。

默认方案：

```text
附加税金 = 销项税额 * 附加税比例
```

如果需要复刻当前表格，也可以使用：

```text
附加税金 = 不含税销售收入 * 附加税比例
```

系统实现建议：

- 参数中提供 `surchargeBase`。
- 可选值：`output_vat`、`revenue_without_tax`。
- 默认值：`output_vat`。

## 7. 利润计算

### 7.1 税前利润

```text
含税金额口径税前利润 = 含税销售收入 - 含税项目全成本
不含税金额口径税前利润 = 不含税销售收入 - 不含税项目全成本
```

### 7.2 所得税

```text
所得税 = max(对应口径税前利润, 0) * 所得税税率
```

说明：

- 税前利润为负时，第一阶段所得税按 0 处理。

### 7.3 税后利润

```text
税后利润 = 对应口径税前利润 - 对应口径所得税
```

## 8. 指标计算

### 8.1 项目毛利率

```text
含税金额口径项目毛利率 = 含税金额口径税前利润 / 含税销售收入
不含税金额口径项目毛利率 = 不含税金额口径税前利润 / 不含税销售收入
```

当不含税销售收入为 0 时，项目毛利率记为 0。

### 8.2 成本利润率

```text
含税金额口径成本利润率 = 含税金额口径税前利润 / 含税项目全成本
不含税金额口径成本利润率 = 不含税金额口径税前利润 / 不含税项目全成本
```

当项目全成本为 0 时，成本利润率记为 0。

### 8.3 销售利润率

```text
含税金额口径销售利润率 = 含税金额口径税后利润 / 含税销售收入
不含税金额口径销售利润率 = 不含税金额口径税后利润 / 不含税销售收入
```

当不含税销售收入为 0 时，销售利润率记为 0。

## 9. 预警规则

建议按项目毛利率判断。

```text
项目毛利率 >= 目标毛利率：优秀
警戒毛利率 <= 项目毛利率 < 目标毛利率：正常
项目毛利率 < 警戒毛利率：警告
```

默认参数：

```text
目标毛利率 = 25%
警戒毛利率 = 15%
```

## 11. 舍入规则

建议：

- 内部计算保留完整精度。
- 页面展示金额保留 0 或 2 位小数，可按用户习惯配置。
- Excel 导出金额建议保留 2 位小数。
- 百分比展示保留 2 位小数。

## 12. 待确认口径

以下口径需要在编码前或第一轮测试时确认：

- 附加税金到底按销项税额计提，还是按不含税销售收入计提。
- 管理费用是否固定按合同收入 6%。
- 期间费用是否全部按合同额计提。
- 所得税税率是否固定，还是按项目或公司主体配置。
- 自有人工是否永远不含税。
