# 实施计划文档

## 1. 实施策略

建议采用小步快跑方式，先完成本地可用的 MVP，再根据实际测算过程补充复杂能力。

实施顺序：

1. 文档和口径确认。
2. 前端项目初始化。
3. 数据模型和计算引擎实现。
4. 页面和交互实现。
5. 本地存储和方案管理。
6. Excel 导出。
7. 样例数据验算。

## 2. 里程碑

### 2.1 M0 文档阶段

交付物：

- PRD 产品需求文档。
- 技术架构文档。
- 数据架构文档。
- 计算规则文档。
- 实施计划文档。

完成标准：

- 明确第一阶段范围。
- 明确主要公式。
- 明确数据模型。
- 明确技术路线。

### 2.2 M1 计算引擎阶段

交付物：

- TypeScript 数据类型。
- 默认参数。
- 计算引擎。
- 格式化工具。
- 单元测试样例。

完成标准：

- 能输入截图表格中的典型数据。
- 能输出收入、成本、利润和利润率。
- 修改参数后结果自动变化。

### 2.3 M2 页面原型阶段

交付物：

- 主测算页面。
- 项目信息表单。
- 参数配置区。
- 收入明细表。
- 成本明细表。
- 指标结果面板。

完成标准：

- 用户可以在页面完成一次完整测算。
- 计算结果实时刷新。
- 金额和百分比格式展示清晰。

### 2.4 M3 方案管理阶段

交付物：

- 方案列表。
- 新建方案。
- 保存方案。
- 复制方案。
- 删除方案。
- 本地存储。

完成标准：

- 关闭浏览器后再次打开仍能看到历史方案。
- 能复制方案做多版本测算。

### 2.5 M4 导出阶段

交付物：

- Excel 导出功能。
- 测算表模板样式。

完成标准：

- 导出的 Excel 包含收入、成本和结果指标。
- 导出结构接近现有测算表。

### 2.6 M5 验算和优化阶段

交付物：

- 样例数据。
- 验算记录。
- 问题修复。

完成标准：

- 使用截图表格对应数据进行核验。
- 关键指标与预期基本一致。
- 已知计算口径差异被记录。

## 3. 第一版页面清单

第一版建议实现两个主要视图：

| 页面 | 说明 |
| --- | --- |
| 测算工作台 | 新建和编辑测算方案 |
| 方案列表 | 查看、复制、删除历史方案 |

如果控制开发量，也可以将方案列表放在测算工作台左侧。

| 组件 | 说明 |

## 5. 第一版开发任务

### 5.1 基础工程

- 初始化 Vite + React + TypeScript。
- 配置基础样式。
- 配置 lint 或基础类型检查。

### 5.2 类型定义

- 定义 Scenario。
- 定义 RevenueItem。
- 定义 CostItem。
- 定义 CalculationParameters。
- 定义 CalculationResult。

### 5.3 计算引擎

- 实现金额不含税换算。
- 实现收入汇总。
- 实现成本明细计算。
- 实现分类成本汇总。
- 实现项目全成本。
- 实现利润和利润率。
- 实现预警等级。

### 5.4 页面实现

- 实现测算工作台布局。
- 实现收入明细编辑。
- 实现成本明细编辑。
- 实现结果面板。
- 实现参数编辑。

### 5.5 本地保存

- 实现 localStorage 读写。
- 实现方案创建、保存、复制、删除。

### 5.6 Excel 导出

- 实现导出数据组装。
- 实现 Excel 文件生成。
- 增加基础表头和格式。

### 5.7 验证

- 使用截图数据创建默认样例。
- 验证收入、成本、利润率。
- 验证导出文件可打开。

## 6. 建议默认样例

根据截图，可预置一个示例方案：

- BIM 建模服务费用：250000 元。
- 框架外包：163600 元，税率 6%。
- 自有人工：3100 元/人天，10 人天。
- 管理费用：按收入 6%。
- 附加税金：按配置公式自动计算。

该样例用于用户首次打开系统时理解操作方式。

## 7. 风险清单

| 风险 | 影响 | 应对 |
| --- | --- | --- |
| 计算口径未完全确认 | 结果可能与 Excel 有差异 | 将差异项参数化，并在文档中标注 |
| 浏览器本地数据丢失 | 历史方案丢失 | 提供导出，后续升级数据库 |
| 导出模板不符合习惯 | 使用体验下降 | 第一版贴近现有 Excel 表 |
| 功能范围膨胀 | 开发周期变长 | 第一阶段只做单项目测算 |

## 8. 编码前待确认事项

建议在正式编码前确认：

- 是否接受 React + TypeScript + Vite 作为第一版技术栈。
- 是否第一版只做本地浏览器保存。
- 附加税金的计提基准。
- 管理费用、销售费用、财务费用是否都按收入比例。
- 导出 Excel 是否要完全复刻现有表格样式。
